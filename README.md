## When MasterNode-IP Address Changed

Just Run these command and follow directions!

**Master Node**
```
git clone https://github.com/dohyunkim12/kubernetes
cd kubernetes
./k8s-masterIP-change.sh
```

<br/>

**Worker Node**
```
git clone https://github.com/dohyunkim12/kubernetes
cd kubernetes
./k8s-masterIP-change-worker.sh
```
